Программа предназначена для просмотра и кластеризации фотографий.
Подробный инструктаж по кластеризации можно посмотреть в подсказках внутри самой программы.
Новые алгоритмы кластеризации легко добавляются в программу, достаточно лишь вставить новый .py файл.
Скрипт должен реализовывать функцию "compare_pictures(first_pictire_path, second_picture_path)"
Функция должна возвращать процент схожести фотографий.
Оба аргумента - полный путь до фотографий.
Этот алгоритм используется каждый раз, когда необходимо сравнить два изображения.
