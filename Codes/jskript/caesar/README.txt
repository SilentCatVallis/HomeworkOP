Производит кодирование шифром Цезаря, а также расшифровку сообщений.

Программа statCreate.js использует текст example.txt для того,
чтобы составить статистику частоты встречаемости букв, и
хранит её в файле stat.txt

Программа caesar.js производит кодирование\декдирование

пример запуска caesar.js:

cscript caesar.js in.txt out.txt decode/encode
in.txt - файл с входными данными
out.txt - файл куда записать результат
decode - расшифровать информацию
encode - зашифровать информацию, после этого ключа обязателен еще один параметр n
[n] - сдвиг, на который нужно сдвинуть буквы в сообщении
