Ищет подстроку в строке алгоритмом Бойер-Мура

пример запуска:
cscript BM.js pattern in.txt [n]
где pattern - подстрока
in.txt - файл в котором происходит поиск
n - необязательный параметр, указывает сколько максимум вхождений выводить на экран

результатом является - время работы, количество вхождений, список начал индексов
