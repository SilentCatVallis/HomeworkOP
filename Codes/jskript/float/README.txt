Float

пример вызова: cscript float.js conv/sum

ключ conv - ковертировать все числа из in.txt в формать float и положить результат в out.txt
ключ sum - конвертировать результат выражения, записанного в постфиксной записи из in.txt в out.txt

все данные записывать в in.txt в отдельной строке. Примеры:
для conv:
	12
	735
	0.235
	-0.2
для sum:
	11
	-0.34
	+
